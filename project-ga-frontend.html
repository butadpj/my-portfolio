Project -
Game-Antena

The purpose

To keep users connected with everything in gaming

The approach

Integrate big platform's services that are related to gaming & streaming

Game-Antena is my first client's project I've ever worked on. My client had already developed the initial website, check it here. My task was to improve every aspects of it and take his existing ideas into a real working app.

After a span of 3 months, the MVP was done and ready to be launch on public. The project is still ongoing as there's still extra requirements to be submitted on Google for the app verification.

Technologies:

  • Frontend - ReactJS, Redux, React Query
  • Backend - NestJS, TypeScript, Docker, PostgreSQL
  • Quality Assurance - Jest, React testing library
