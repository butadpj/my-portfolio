Project -
Game-Antena (backend)

The purpose

To provide seamless data processing and API handling for Game-Antena's frontend

The approach

Develop simple and easy to use API endpoints that the frontend would use

Developing Game-Antena's API (backend) has brought me lot of challenges to face. First was the 3rd party API services integration. It was hard at first since it was my first time doing integration. But my years of expertise on reading documentations and self learning has helped me a lot.

The next challenge was keeping the backend maintainable and scalable. I've learned that it is better to do things correctly at first than to suffer from technical debts on the future.

Those are just simple challenges I've faced but I'm still glad I was able to experience those. Those only made me a better developer than I am years ago

Technologies:

  • Language/Tools - NestJS (TypeScript), Docker
  • Database - PostgreSQL, TypeORM
  • Quality Assurance - Jest
  • Hosting - Heroku
