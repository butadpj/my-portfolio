Project -
Modern Web Dev

The purpose

My goal was to generate a multi-page web app boilerplate like create-react-app

The approach

Create small separate packages to be used in create-modern-web-dev

Package #1

Utility package that will be used in webpack configuration - modern-web-dev-utils

Package #2

Package to contain webpack configuration and package.json scripts - modern-web-dev-scripts

Developing static websites with multiple pages (like this website) don't need the complexity of modern frameworks like React, Vue, or Angular. But maintaining websites built with plain HTML, CSS, and JS without structure is difficult as well.

That's why Modern Web Dev is born. It brings structure to the messy development environment "vanilla" brings. You don't have to learn another framework just to build simple static websites for your personal or business needs.

Easily setup Modern Web Dev on your own with this simple command
